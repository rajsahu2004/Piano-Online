Piano Online

A simple paino Website which has the following functions -

  • A sign-in-signup landing page for user authentication.
  • Users should be able to play the piano using click and also keys.
  • The user should start the timer by clicking on a button when he plays and stop it before leaving.
  • Display the total time played on the main page.
  • Can store particular music notes created only when the user is logged in.
